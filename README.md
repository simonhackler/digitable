# Digitable

Digitable is a free and open source board game engine.

[![Discord](https://img.shields.io/badge/Discord-Join%20Chat-5865F2?logo=discord&logoColor=white)](https://discord.gg/sasxJ5MRWX)

Try it out at: https://digitable.ink/

## Design philosphy

- Open source. There are a lot of reasons for this. Me personally I only use open source software and want board game design software to be open as well.
- (almost) Everything is files. A great writeup of this philosophy: https://stephango.com/file-over-app
- Made for everyone. Digitable should be used by designers creating their first prototype and professionals working on their 100th game.
- Support the whole game design lifecycle. The goals is to be able to create the first prototype and continously refine it in digitable until it is sent off to the printer.
